# 🎵 MusicStream Web App

MusicStream is a **simple, lightweight web application** for streaming music using **HTML, CSS, and JavaScript**. Users can play, pause, skip tracks, and enjoy a basic playlist experience directly in the browser.

---

## Features

- Play, pause, and skip songs
- Simple playlist management
- Show current track info
- Responsive design for desktop and mobile
- Light and dark mode support

---

##  Tech Stack

- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Audio Playback:** HTML5 `<audio>` element
- **Responsive Design:** CSS Flexbox/Grid
- **Hosting:** Any static server (GitHub Pages, Netlify, Vercel)
